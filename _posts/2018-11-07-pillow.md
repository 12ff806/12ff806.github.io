## 简介 ##

用Pillow进行图片处理

## 安装 ##

~~~sh
pip3 install pillow
~~~

## 简单操作图片 ##

包括对图片缩放，粘贴，画图等操作

~~~python3
from PIL import Image, ImageFont, ImageDraw, ImageFile

ImageFile.LOAD_TRUNCATED_IMAGES = True    # 解决 "IOError: image file is truncated" 问题


img1 = Image.open("test.jpg")    # 打开图片

img2 = Image.open("test.png")    # 打开png图片

\*drop, alpha = img2.split()    # 将png图片粘贴到背景图上
img1.paste(img2, (0, 0), mask=alpha)

img3 = Image.new("RGB", (300, 300), (255, 255, 255))    # 创建新图片

img3 = img3.crop((20, 20, 280, 280))    # 剪切图片
img3.load()

img3.thumbnail((200, 200))    # 按比例缩放

img3 = img3.resize((150, 150), Image.ANTIALIAS)    # 缩放, 设置Image.ANTIALIAS参数为抗锯齿

img3_dr = ImageDraw.Draw(img3)    # 获取ImageDraw对象来画图

font = ImageFont.truetype('/Library/Fonts/Arial.ttf', 20)
img3_dr.text((10, 10), "hello world", font=font, fill="#000000")    # 写字

img3_dr.line((10, 30, 140, 30), fill="#000000", width=2)    # 画线

img3.save("hw.jpg", quality=100)    # 保存图片
~~~
